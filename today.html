Today we're doing: building websites!

This (and the main page) was written with HTML and CSS! We talked about it last week. HTML stands for Hyper Text Markup Language and represents the content and structure of a web page. Content is created using tags. CSS stands for Cascading Style Sheets and helps style a web page. Inspect this website to see the styles for it!
